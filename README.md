## Команды
#### Перейти в директорию server и запустить сервер
####
```sh
npm i - установить зависимости
```
####
```sh
npm start - запустить сервер
```
#### Перейти в директорию client и запустить react-приложение.
####
```sh
npm i - установить зависимости
```
####
```sh
npm start - запустить приложение
```
